# 🎵 Edutreble

**Plataforma Web Interactiva para el Aprendizaje Musical Gamificado**

> Un proyecto educativo que transforma la enseñanza musical mediante gamificación, diseño interactivo y entornos virtuales accesibles para jóvenes.

---

## 📌 Descripción

Edutreble es una plataforma web estática desarrollada con HTML, CSS y JavaScript que busca resolver uno de los mayores problemas de la educación musical virtual: la falta de motivación y el abandono temprano. Mediante mecánicas de juego (gamificación), lecciones interactivas y retroalimentación inmediata, Edutreble convierte el aprendizaje musical en una experiencia atractiva y sostenida.

El proyecto surgió como respuesta a datos preocupantes: el **77% de los jóvenes encuestados no estudia teoría musical regularmente**, mientras que el **66.7% sí estaría dispuesto a aprender mediante sistemas tipo juego**.

## ✨ Funcionalidades

- 🏠 **Página de inicio** con menú interactivo y modo claro/oscuro
- 📚 **Módulo de cursos** con lecciones progresivas por nivel
- 🎮 **Minijuego educativo interactivo** (identificación de notas musicales)
- 🔊 **Audio real** de cada nota musical (Do, Re, Mi, Fa, Sol, La, Si)
- ⚡ **Retroalimentación inmediata** ante respuestas del usuario
- 👥 **Sección comunidad** para interacción entre estudiantes
- 🌗 **Modo oscuro / claro** toggle

## 📖 Fundamentos teóricos

El proyecto se apoya en tres pilares académicos:

- **Teoría de Autodeterminación** (Deci & Ryan): la motivación intrínseca florece cuando se satisfacen autonomía, competencia y relación social.
- **Aprendizaje Significativo** (Ausubel): integrar teoría y práctica en contextos coherentes y conectados con los intereses del estudiante.
- **Gamificación efectiva**: puntos, insignias, niveles y retroalimentación inmediata incrementan el compromiso sostenido.
